# Schema Reference — bc-ai-directory-public-v1

## JSON Structure

```
{
  schemaVersion: "bc-ai-directory-public-v1",
  source: { ... },
  counts: { people: {...}, organizations: {...}, records: {...} },
  people: [ PersonRecord, ... ],
  organizations: [ OrgRecord, ... ]
}
```

---

## Organization Record

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique identifier: `"organization:{slug}"` |
| `type` | string | Always `"organization"` |
| `slug` | string | URL-safe unique identifier (kebab-case) |
| `name` | string | Display name |
| `summary` | string | Public-safe summary, max ~420 characters |
| `excerpt` | string | Shorter excerpt for card views, max ~180 characters |
| `categories` | string[] | Classification tags (e.g. `["AI/ML", "Health Tech"]`) |
| `locations` | string[] | BC locations (e.g. `["Vancouver", "Victoria"]`) |
| `tags` | string[] | Topic and keyword tags |
| `websiteUrls` | string[] | Public website URLs (HTTPS only) |
| `profileUrl` | string | Path to public detail page on bc-ai.ca portal |
| `lastUpdated` | string | ISO 8601 date of last profile update |

---

## Person Record

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique identifier: `"person:{slug}"` |
| `type` | string | Always `"person"` |
| `slug` | string | URL-safe unique identifier (kebab-case) |
| `name` | string | Display name |
| `summary` | string | Public-safe summary, max ~420 characters |
| `excerpt` | string | Shorter excerpt for card views, max ~180 characters |
| `roles` | string[] | Professional roles (e.g. `["Founder", "Researcher"]`) |
| `locations` | string[] | BC locations |
| `organizations` | string[] | Affiliated organizations (names, not slugs) |
| `tags` | string[] | Topic and keyword tags |
| `profileUrl` | string | Path to public detail page on bc-ai.ca portal |
| `lastUpdated` | string | ISO 8601 date of last profile update |

---

## CSV Encoding

Array fields are serialized as **pipe-separated strings** in the CSV files:

```
"AI/ML|Health Tech|Nonprofit"
```

To parse in Python:
```python
df['categories'] = df['categories'].str.split('|')
```

To parse in JavaScript:
```js
const categories = row.categories ? row.categories.split('|') : [];
```

Empty arrays are encoded as empty strings `""`.

All string fields are double-quoted. Internal double-quotes are escaped as `""` (RFC 4180 compliant).

## Counts Object

```json
{
  "people": {
    "total": 1454,
    "exported": 524,
    "withheld": 930,
    "withheldReasons": {
      "nonPublic": 873,
      "incomplete": 24,
      "unsafe": 33
    }
  },
  "organizations": {
    "total": 953,
    "exported": 865,
    "withheld": 88,
    "withheldReasons": {
      "nonPublic": 18,
      "incomplete": 64,
      "unsafe": 6
    }
  }
}
```

`withheldReasons`:
- `nonPublic` — profile exists but is marked non-public by the subject
- `incomplete` — missing required fields (name, summary)
- `unsafe` — content flagged during safety review
